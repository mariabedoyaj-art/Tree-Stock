# Tree-Stock
## Objetivo
Desarrollar un sistema de inventario usando un árbol binario de búsqueda en Java.

## Funcionalidades
- Registrar productos
- Mostrar inventario ordenado
- Buscar productos por ID

## Estructura del proyecto
- Producto.java → Nodo del árbol
- ArbolInventario.java → Lógica del árbol
- Main.java → Menú interactivo
